# MakeTexChigga

Веб-редактор отчётов с генерацией готового `.tex` файла под шаблон ДВГУПС. Проект заменяет старый монолитный `oldsite.htm`: теперь титульный лист, разделы, блоки текста, рисунки, код, таблицы и списки редактируются в нормальном широком интерфейсе.

> Проект полностью сделан в нейронке.

## Что умеет

- редактирование титульного листа;
- добавление разделов уровней `section`, `subsection`, `subsubsection`;
- блоки текста, рисунков, кода, таблиц, списков и разрывов страниц;
- блоки графиков, которые рисуются самим LaTeX через `pgfplots`;
- перемещение разделов и блоков вверх/вниз;
- автосохранение черновика в браузере;
- сохранение и загрузка проекта через `.json`;
- генерация полного LaTeX-документа;
- копирование `.tex`, скачивание `report.tex`, локальная сборка `PDF` и быстрый переход в Overleaf.

## Запуск

```bash
npm install
npm run dev
```

После запуска открой:

```text
http://127.0.0.1:3000
```

## Сборка

```bash
npm run build
```

## Локальная сборка PDF

Кнопка `Скачать PDF` работает через локальный `pdflatex`.

Нужно установить один из вариантов:

- MiKTeX
- TeX Live

Если компилятор не установлен, приложение покажет понятную ошибку и предложит поставить его.

## Как пользоваться

1. Заполни титульный лист.
2. Добавь разделы и нужные блоки.
3. Нажми `Сохранить проект`, чтобы выгрузить редактируемый черновик в `.json`.
4. Позже нажми `Загрузить проект` и выбери этот `.json`, чтобы продолжить работу.
5. Нажми `Сгенерировать .tex` или просто скопируй автоматически обновляемый результат.
6. Открой Overleaf по кнопке в интерфейсе.
7. Вставь содержимое `.tex` в новый проект.
8. Если есть рисунки, добавь рядом папку `images/` и положи туда файлы с теми именами, которые указаны в блоках рисунков.

## Структура проекта

- `app/page.tsx` — основной интерфейс редактора.
- `app/globals.css` — стили тёмной темы.
- `lib/report.ts` — модель отчёта и генератор LaTeX.
- `oldsite.htm` — старая версия, оставлена для истории и сверки поведения.
